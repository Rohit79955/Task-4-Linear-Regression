# 🏡 Task 4: Linear Regression Modeling

📘 **Data Science Internship - EduTech Solutions**

---

### 🎯 Objective
To understand machine learning fundamentals by building, training, and evaluating a supervised Linear Regression model to predict property values based on house dimensions.

---

### 📊 Model Evaluation Results
* **Features Trained:** House Size (SqFt)
* **Target Output:** House Price
* **Evaluation Metric:** Root Mean Squared Error (RMSE) was calculated to confirm average predictive variation.

---

### 🛠️ Tools Used
* Python
* Pandas & NumPy
* Scikit-Learn (Model Selection & Linear Model)
* Matplotlib (Visualization)
